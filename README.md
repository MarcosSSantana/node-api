# Node-api

API em NodeJs utilizando Typescript, Express e Sequelize

### Comandos

- Installation
  ```javascript
  npm init
  ```

- Run project
  ```javascript
  npm ren dev
  ```

- Up DB
  ```javascript
  npx sequelize db:create
  ```

- Create migrations
  ```javascript
  npx sequelize-cli migration:generate --name {NOME}
  ```

- Create tables
  ```javascript
  npx sequelize db:migrate
  ```

- Run build 
  ```javascript
  npm ren build
  ```
